# .brain - Project Memory

This folder contains project documentation and development logs for the Lapland Famille website.

## Files

- **project-log.md** - Complete development log including:
  - Project overview and tech stack
  - All components created
  - Data structures
  - SEO implementation
  - Deployment history
  - Git commits
  - Future integration notes

## Quick Reference

**Live Site:** https://laplandfamille.vercel.app  
**GitHub:** https://github.com/ani-shh/laplandfamille  
**Tech Stack:** Next.js 14 + Tailwind CSS + TypeScript + Framer Motion

## Key Commands

```bash
# Development
npm run dev

# Build
npm run build

# Deploy to Vercel
vercel --prod

# Push to GitHub
git push origin main
```
